### Hi there 👋

Here are some ideas to get you started:

* This is a springboot application developed with JDK 17
* run mvn clean install
* start the application using the base class AnswertjianeApplication
* after the application has started, navigate to http://localhost:8080/swagger-ui/index.html on your browser
* APIs can be called directly on swagger-ui
* Please refer to data.sql for test data to test the application
